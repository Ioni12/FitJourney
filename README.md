# FitJourney

FitJourney is a comprehensive application designed to help users track their fitness, nutrition, and wellness goals. Built with a modern technology stack, it features a robust backend for data management and a dynamic frontend for an intuitive user experience.

## Project Structure
This repository contains two main directories:
* `backend/`: The server-side application providing the API and data storage solutions.
* `frontend/`: The client-side application providing the user interface.

## Quick Start

### Prerequisites
* Node.js (v16+)
* npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ioni12/FitJourney.git
   cd FitJourney
   ```

2. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```

3. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```

### Running the Application

1. Start the backend server:
   ```bash
   cd backend
   npm start
   ```

2. Start the frontend development server:
   ```bash
   cd ../frontend
   npm start
   ```
